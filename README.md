# Análise de Dados do Campeonato Brasileiro de Futebol

Este projeto realiza uma análise exploratória e visualização de dados do Campeonato Brasileiro de Futebol utilizando Python. O objetivo é explorar padrões de vitórias, distribuição de gols e estatísticas de desempenho ao longo das rodadas.

## Funcionalidades

- Carregamento e limpeza de dados diretamente do Kaggle
- Preparação dos dados e remoção de colunas desnecessárias
- Análise descritiva dos resultados dos jogos:
  - Distribuição de vitórias por mandante, visitante e empates
  - Estatísticas de gols marcados por mandantes e visitantes
  - Média de gols por rodada
- Visualizações avançadas:
  - Histogramas de gols
  - Boxplots comparativos
  - Heatmaps de frequência de placares
  - Gráficos de linha da evolução das médias de gols
  - Gráficos de pizza mostrando proporções de resultados

## Tecnologias Utilizadas

- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Seaborn
- KaggleHub (para carregar datasets do Kaggle)
